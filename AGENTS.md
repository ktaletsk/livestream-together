# AGENTS.md

This repo tracks prep notes for the "livestream together" series.
The show is a friendly, cross-ecosystem livestream/podcast where guests analyze the same dataset for ~1 hour using their own tools, then compare approaches and results.

Hosts (bios are short, public-profile based):
- Konstantin Taletskiy (you): Jupyter OSS developer at Anaconda and founder of orbrx, working in the Jupyter ecosystem and open source tooling.
- Rodrigo Silva Ferreira: QA engineer at Posit PBC focused on quality/usability of open-source R and Python tooling; speaker at PyData Boston 2025 on large-scale water quality analysis using USGS APIs and Positron.
- Yann Debray: MATLAB Online product manager at MathWorks and author of "MATLAB with Python," focused on MATLAB and Python interoperability and GitHub workflows.
- Jake Diamond-Reivich: Founder of Mito (Mito AI / Mito Labs), building spreadsheet-to-Python tooling; Project Jupyter committee member and Wharton BS in Finance.

- Episodes live in top-level folders named `DD-MM-YY`.
- Each episode folder should include:
  - `BRIEF.md`: 3-6 bullets with dataset name/source and the goal.
  - `NOTES.md`: optional working notes during analysis.
  - `RESULTS.md`: optional post-episode summary.

Keep files short, ASCII-only, and prefer bullets.

TidyTuesday approach (from the upstream readme):
- Data posted weekly; use the week instructions or download from GitHub.
- Explore relationships; avoid causal claims.
- Produce something shareable (viz/model/report/app).
- Share results with #TidyTuesday (and tool-specific tags if relevant).
